Phase 1: Develop a fully coupled monolithic solver

First phase is to make a fully monolithic solver which will yield accurate results at the cost of speed. For verification of implementation we will use Turek as reference. We will explore and use the different testcases derived in this paper for comparison of our Fluid, Structure and FSI implementation. As an extension we will consider both Arbitrary Lagrangian Eulerian and a fully Eulerian description for the FSI models.

Main strategy to overcome numerical errors in CSM3

  • Ensuring the numerical implementations are centred correctly
  • Handling of the convection term in the Lagrangian formulation, when calculating deformation.
  • Implementation of alternative numerical schemes:
    • Frac-step
    • Newmark
    • Adam-bashford Linear multistep methods
